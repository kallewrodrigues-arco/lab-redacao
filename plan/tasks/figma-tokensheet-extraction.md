# Task: figma-tokensheet-extraction

## Escopo
- **Tipo**: Front-end (FE)

## Objetivo
Extrair uma **tokensheet** do Figma (cores, tipografia, spacing, radius, shadows, grid) e mapear para o tema do projeto (ex.: CSS variables e/ou Tailwind config) para garantir consistência visual no protótipo.

## Entregáveis
- Tokens consolidados (nomes + valores + usos previstos).
- Mapeamento para tema do projeto (ex.: `:root` CSS vars / `tailwind.config` / theme provider).
- Convenção de nomes (ex.: `color.bg.surface`, `space.4`, `radius.md`, `shadow.sm`).

## Critérios de pronto
- Tokens cobrem os elementos usados nas telas do piloto.
- Não há “valores mágicos” recorrentes no UI (cores/spacing/radius) fora do tema.
- Não há valores identicos guardados em 2 tokens diferentes.
- Existe uma fonte única de tokens para o protótipo.
- Existe um documento chamado tokens.htmlna pasta assets, onde é possível ver os seguintes elementos: Headers, Parágrafo, Inputs, Captions, Buttons e Divs
